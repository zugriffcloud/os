zugriff, a WeltByte GmbH product, is the uncomplicated hosting solution at the Edge, close to your users, for
modern [JavaScript Frameworks](/ecosystem/integrations) and [Static Web Applications](/guides/static-web-applications)
Europe deserves. All without compromising on GDPR, developer experience, or performance.

Deploy your application using the opt-in GitHub integration, the command-line interface, or the well-documented
[API](/reference/openapi), and collaborate with confidence using role-based access controls built on zero-trust
principles.

To server-side-render your requests as efficiently as possible, we use our own [JavaScript Runtime](/reference/runtime-apis)
as an alternative to [NodeJS](https://nodejs.org). This allows us to almost eliminate cold-starts and better parallelise execution.  
Using isolates, sandboxed JavaScript execution environments, we protect your execution environment
from others we host.

Learn more about [framework integrations](/ecosystem/integrations),
[request and response streaming](/reference/runtime-apis/#readablestream), [regional deployments](/concepts/dns),
[supported APIs](/reference/runtime-apis), [Node.js compatibility](/reference/runtime-apis/#nodejs-compatibility),
[establishing TCP connections](/reference/runtime-apis#sockets-tcp) (Cloudflare- and Node.js-compatible), and our ecosystem of
vendor-lock-free connection-pooling proxies to send [emails](/ecosystem/addons/smtp) and communicate with
[databases](/ecosystem/addons/postgres) or [key-value stores](/ecosystem/addons/redis).

Built on open standards, we believe in earning your business, not forcing it.
